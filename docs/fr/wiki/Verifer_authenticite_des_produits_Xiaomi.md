## Vérifier l'authenticité des produits Xiaomi

### Téléphones

Vérification via IMEI et numéro de série [ici](https://www.mi.com/global/verify/#/en/tab/imei). (en)

* Note : malgré ce que dit le site, le numéro de série ne peut être trouvé que sur l'autocollant de code qui se trouve au dos de la boîte d'emballage. Si quelque chose ne va pas, même si ce n'est pas l'IMEI, le site dira que l'IMEI est erroné, veuillez garder cela à l'esprit.

### Produits audio

La plupart des produits audio de Xiaomi sont livrés avec un code QR unique qui vérifie leur authenticité en pointant vers un site web de vérification, mais ce site web apparaît souvent vide lorsqu'on y accède en dehors de la Chine, il faut donc jouer avec l'URL pour qu'il montre le résultat.

L'URL que le code QR contient :

[https://www.mi.com/verify/hsyz/?serial](https://www.mi.com/verify/hsyz/?serial) (en)

L'URL que vous devez visiter pour obtenir le résultat :

[http://auth.1more.com/checkSN.html?c=serial&lag=en](http://auth.1more.com/checkSN.html?c=serial&lag=en) (en)

*Note : 1MORE est une société audio chinoise dans l'écosystème Mi.*

### Tout le reste

Vérification par code de sécurité [ici](https://www.mi.com/global/verify/#/en/tab/secur). (en)
